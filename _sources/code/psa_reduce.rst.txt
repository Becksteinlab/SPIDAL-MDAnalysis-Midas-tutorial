===============
 psa_reduce.py
===============

The :file:`rp/psa_reduce.py` implements the *reduce* step by combining
the partial results from :doc:`mdanalysis_psa_partial` into the full
PSA distance matrix. It also contains analysis code to cluster and
plot the matrix.
