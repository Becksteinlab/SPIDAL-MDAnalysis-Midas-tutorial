SPIDAL Tutorial: MDAnalysis with Midas/radical.pilot¶

version:1.2
date:Apr 21, 2017
repository:https://github.com/MDAnalysis/SPIDAL-MDAnalysis-Midas-tutorial

The tutorial will showcase how to use Midas with radical.pilot to parallelize the analysis of an ensemble of macromolecular transition paths with Path Similarity Analysis (PSA, [Seyler2015]) as implemented in the MDAnalysis library.
